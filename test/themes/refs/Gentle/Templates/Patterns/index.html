Escape unit test

Tuesday 13. August 2019

As shown below Escape will disable parsing of both VenC patterns and markup language between the Escape/EndEscape pair.

With Escape/EndEscape

- Hello VenC! - Hello lovely user!

.:SetColor::This is red text!t::red:.

Without Escape/EndEscape

  • Hello VenC!
  • Hello lovely user!

This is red text!t

Please note that VenC syntax require you to close Escape pattern with ::EndEscape.
